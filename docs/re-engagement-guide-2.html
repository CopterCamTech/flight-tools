ğŸ“‚ Why BIN and ULG Are Separated

Flight-Tools supports two distinct UAV log formats:

  • .bin (ArduPilot): parsed using pymavlink
  • .ulg (PX4): parsed using pyulog

Each format requires a different parser, message structure, and diagnostic strategy. To maintain clarity and reproducibility, Flight-Tools separates these workflows into parallel script sets and route handlers. This avoids conditional logic clutter and makes it easier to extend or debug each format independently.
